SSN

  • inprogress

This is an SSN input.

<div class="sprk-b-InputContainer">
  <label for="ssn">Social Security #</label>
  <input class="sprk-b-TextInput sprk-u-Size-100 js-ssn" id="ssn" type="text" placeholder="000-00-0000">
  <div class="sprk-b-SelectionContainer">
    <input id="show-ssn" type="checkbox" class="js-remove-mask">
    <label for="show-ssn" class="sprk-b-Label--inline sprk-u-mls">Show SSN</label>
  </div>
</div>

Additional Info

  • Use *'s to mask the value by default.
  • Checking "Show SSN" removes the mask and shows the actual SSN.
  • On blur of the text input, if the "Show SSN" box is checked, add dashes between the 3rd and 4th number and the 5th and 6th number (ex. 000-00-0000).
  • Placeholder text should read: 000-00-0000.
  • On key press, reveal the placeholder inline with the label.

      <span class="">8 or more characters</span>
    </li>
    <li class="sprk-c-ValidationCriteria__item sprk-c-ValidationCriteria__item--unmet">
      <svg class="sprk-c-Icon" viewBox="0 0 100 100">
        <!-- href should point to the icon svg in your project -->
        <use xlink:href="../../assets/toolkit/images/icons.svg#times" />
      </svg>
      <span class="">Upper and lowercase letters</span>
    </li>
    <li class="sprk-c-ValidationCriteria__item sprk-c-ValidationCriteria__item--unmet">
      <svg class="sprk-c-Icon" viewBox="0 0 100 100">
        <!-- href should point to the icon svg in your project -->
        <use xlink:href="../../assets/toolkit/images/icons.svg#times" />
      </svg>
      <span class="">Use a number</span>
    </li>
  </ul>
</div>

Additional Info

  • This is only used for fields that have uncommon criteria or restrictions. For example, password and username.
  • The criteria should start hidden and appear below the form field when the field is focused.
  • Once anything is entered into the field, the criteria should remain visible.
  • On first focus, the criteria are gray and will become green as they are met.
  • As long as it is the first time you are focusing the field nothing will turn red.
